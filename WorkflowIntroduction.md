# n8n 工作流程簡介：My workflow.json

這是一份 **n8n (自動化工作流程)** 的匯出檔（`My workflow.json`），主要運作邏輯是透過 Webhook 接收 **LINE Bot** 回傳的訊息，並根據不同的訊息類型與關鍵字，自動分流執行 **四個主要功能**：

### 1. 🍽️ 餐廳爬蟲與 AI 美食推薦信件
- **觸發條件**：使用者文字訊息包含「好評推薦：」。
- **執行流程**：
  - 呼叫 **Apify** 的 Google Maps 爬蟲（`crawler-google-places`）去網路上抓取指定地點或餐廳的資料。
  - 獲取並彙整（Aggregate）這些餐廳的評分、地址、圖片等資訊。
  - 將資料交給 **Google Gemini (LLM)**，並賦予其「美食雜誌編輯」的人設，請它分析數據並篩選出前 10 名近期最熱絡的餐廳，自動排版成精美的 HTML 左圖右文格式。
  - 將做好的電子報透過 **Gmail** 寄送給指定信箱，最後透過 LINE 提示使用者「✅ 您的美食研究報告已寄出！」。

### 2. 📂 多媒體檔案自動上傳雲端
- **觸發條件**：使用者傳送「圖片 (image)」、「影片 (video)」或「檔案 (file)」。
- **執行流程**：
  - 針對傳送來源（個人 ID 或特定群組 ID）判斷並對應到設定好的 **Google Drive 資料夾 ID**。
  - 透過 LINE API 下載該檔案的內容。
  - 將檔案自動上傳至對應的 **Google Drive** 目錄中。
  - 上傳成功後，回傳雲端硬碟的分享連結給使用者。

### 3. 🎙️ 語音訊息 AI 轉文字 (逐字稿)
- **觸發條件**：使用者傳送「語音 (audio)」訊息。
- **執行流程**：
  - 進行權限驗證（僅限直接對話或白名單群組可用）。
  - 下載語音檔案的內容。
  - 將音檔傳送給 **Google Gemini (`gemini-2.0-flash`模型)** 將語音逐字轉錄為繁體中文。
  - 將語音轉錄的文字成果直接透過 LINE 回覆給使用者。

### 4. 🆔 LINE ID 快速查詢器
- **觸發條件**：使用者輸入 `id`、`getid` 或 `查詢id`。
- **執行流程**：
  - 直接抓取並回覆觸發該次對話的 **Group ID（群組）** 或 **User ID（個人）** 給使用者，通常用於開發者後台配置群組白名單與權限時使用。

---
📝 **總結**：
這是一個非常實用且整合度高的 LINE 智慧助理工作流程。它完美串接了 **LINE 互動、Apify 網路爬蟲、Google Gemini 生成式 AI 與語音辨識、Gmail 寄信以及 Google Drive 儲存** 等多項雲端服務。
